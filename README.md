# piscine-ocaml

```

Learn OCaml in 2 intensive weeks !

        _
    .--' |
   /___^ |     .--.
       ) |    /    \
      /  |  /`      '.
     |   '-'    /     \
     \         |      |\
      \    /   \      /\|
       \  /'----`\   /
       |||       \\ |
       ((|        ((|
       |||        |||
      //_(       //_(

```

Run the exercices in the interpreter : 
```
ocaml <file>
```

Compile it in native :
```
ocamlopt <file>
./a.out
```

Compile it in bytecode :
```
ocamlc <file>
./a.out
```

... or "make" if there is a Makefile !
